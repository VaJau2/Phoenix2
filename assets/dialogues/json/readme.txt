Как читать диалог (пособие для юных скриптов):

- грузим диалоги по циклу по порядковому номеру
- левые и правые пути картинок парсим из "left" и "right"
	- если есть путь со словом strikely, то в конец пути надо добавить название расы и ".png"
	- иначе просто добавить ".png"
- если есть "text", значит облако не нужно, текст в менюшке снизу, а этот нод можно пропустить пробелом
	- иначе берем текст из phrase, положение берем из первого его значения, а текст из второго
		- варианты положений:
			left - стрелка на левого собеседника
			right - стрелка на правого собеседника
			top - стрелка на левого(в другом смысле) собеседника
			none - без стрелки

- если есть "effects", то эффект нужно взять из списка эффектов и присвоить ему новое значение
- если "effects" нет, то обнуляем все наложенные раннее эффекты
	- эффекты из списка надо обрабатывать в _process, проходя по ним циклом, проверяя значение и изменяя соотстетствующую штуку

- если есть tonode, следующий нод берется из него
	- иначе следующие ноды надо брать из второго значения массивов внутри answers

----------------------------------------------
список эффектов:

left[/right]-shake-x - картинка дрожит по горизонтали постоянно, скорость плавно меняется до нового значения
left[/right]-shake-x-once - картинка дрожит по горизонтали один раз, скорость меняется резко 
left[/right]-shake-y - картинка дрожит по вертикали постоянно, скорость плавно меняется до нового значения
left[/right]-shake-y-once - картинка дрожит по вертикали один раз, скорость меняется резко
